# TodoList
A simple ASP.NET Core 6 Blazor todo application.

## Prerequisites
1.  `dotnet` 6 CLI installed.  CLI is included with .NET Core, so make sure that is [installed](https://docs.microsoft.com/en-us/dotnet/core/install/)

## Getting Started
1. Clone this repo
2. Restore project
```
dotnet restore
```

## Clean
```
dotnet clean
```

## Build
```
dotnet build
```

## Test
```
dotnet test
```
